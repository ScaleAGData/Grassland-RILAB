GPP Estimation from Sentinel-1 (S14GPP)

This repository contains the training pipeline and inference tools for estimating Gross Primary Production (GPP) using Sentinel-1 (SAR) satellite data. This work is part of the ScaleAgData initiative (WP5 - Grassland RILAB).  
Overview
Unlike optical sensors, Sentinel-1's Synthetic Aperture Radar (SAR) can penetrate cloud cover, providing a consistent 
temporal signal of grassland structure. This module uses backscatter coefficients (VH and VV) combined with temporal features to model carbon fluxes.  Repository StructureTraining/: Contains the core logic for model development.  ANNmodel_GPP_sentinel1_9.py: The training script for the Multilayer Perceptron (MLP) neural network.  IFAPA_merged_S1_IFAPAData2024_input.xlsx: The training dataset containing Sentinel-1 features and ground-truth Eddy Covariance (EC) GPP data.  training_outputs/: Artifacts generated during the training sessions.  ECGPP_best_model_*.keras: The trained neural network weights.  scaler_X.pkl / scaler_y.pkl: Normalization parameters for inputs and targets.  ECGPP_results_*.txt: Performance logs (R², RMSE, RAE).  GeospatialGPPInference.py: The script used to apply the trained MLP model to GeoTIFF rasters to generate GPP maps.  Technical WorkflowFeature Engineering: The model utilizes $VH_{av}$ and $VV_{av}$ backscatter, along with a calculated Day of Year (DOY) feature to capture seasonal growth cycles.  Normalization: Features are standardized using a StandardScaler, while the GPP target is scaled to a $[0, 1]$ range using a MinMaxScaler.  Optimization: The training script executes an outer loop (up to 10,000 iterations) to find the optimal random weight initialization, maximizing the Validation R².  Inference: The inference script flattens input GeoTIFFs, masks NoData pixels, performs the prediction, and reconstructs a georeferenced GPP map.  How to Run1. Environment SetupEnsure you have created the Conda environment using the environment.yml file located in the root directory.  Bashconda activate gpp_env
2. TrainingTo retrain the model using the IFAPA dataset:  Bashpython Training/ANNmodel_GPP_sentinel1_9.py
3. InferenceTo generate a GPP map from a Sentinel-1 GeoTIFF:  Update the input_tiff and model_path variables in GeospatialGPPInference.py.  Execute the script:Bashpython GeospatialGPPInference.py
Metrics and PerformanceThe models are evaluated using the following statistical indicators:  R² (Coefficient of Determination)RMSE (Root Mean Squared Error)RRMSE (Relative RMSE)RAE (Relative Absolute Error)For detailed performance results of the latest run, refer to the .txt and .xlsx files within the training_outputs folder.
